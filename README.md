## Installation & Run Locally

### 📦 Prerequisites
- Node.js installed on your system

---

### 💻 Setup Steps

1. Install dependencies:
```bash
npm install
````

2. Create `.env.local` file and add your Gemini API key:

```
GEMINI_API_KEY=your_api_key_here
```

3. Run the development server:

```bash
npm run dev
```

4. Open in browser:

```
http://localhost:3000
```
